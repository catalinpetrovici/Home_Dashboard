# Home Dashboard

- Dashboard for home

- Features
	-	users can sign up, log in, log out
	-  users can do CRUD operations

-   Backend Framework
	-	Node.js with Typescript

-   Frontend
	-	React with Typescript
	
-   Database
	-	Redis
	- PostgreSQL
